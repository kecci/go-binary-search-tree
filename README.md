# binary search tree (BST) in go

- 

## Definitions
binary tree is 
- tree data structure
- each node has at most two children

binary search tree (BST) is
- rooted binary tree
- internal nodes store a key

## Why use this
Runtime
- O(log n)
  - Insert
  - Remove
  - Search

Space
- O(n)

## Implementing In Go
- node and BST wrapper
- String()
- Add()
- Search()
- Remove()

## Node & BST Wrapper
- Node
  - Value
  - Left
  - Right
- BST Wrapper
  - Root
  - Len
